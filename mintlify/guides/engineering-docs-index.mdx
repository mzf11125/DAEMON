## Workflow

1. Edit authoritative content in `docs/*.md` when doing deep engineering updates.
2. Port summaries or full content into matching `mintlify/**/*.mdx`.
3. Run `cd mintlify && npx mint validate && npx mint broken-links` before merge.

Use [Mintlify MCP](/guides/using-mintlify-mcp) for component syntax and deploy settings.
